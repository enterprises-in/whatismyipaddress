## Run Locally

Clone the project

```bash
  git clone https://github.com/enterprises-in/whatismyipaddress.git
```

Go to the project directory

```bash
  cd whatismyipaddress
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run dev
```
